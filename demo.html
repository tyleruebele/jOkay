Demo for validate-phone.js

validate-phone.js is a pure JavaScript (no dependencies) solution to make HTML phone inputs self-validating.

Basic Usage

The basic phone validation functionality can be added simply by including validate-phone.js anywhere in your document and adding CSS class 'js-validate-phone' or type 'phone' to all inputs you want validating. It works like this:


Stricter / Looser Usage

By default, phone numbers are validated in a loose NANPA format. A short list of rules is checked. Phones can be validated against a looser or stricter format. To accept letters for phonewords, add CSS class 'js-validate-phone-wordy' to the input. Likewise, to enforce known area codes, apply css class 'js-validate-phone-strict' to the input. When using strict testing, it is important to frequently update the list of area codes.


Styling Errors

Inputs with invalid phones can be styled by applying styles to the error class. By default this class is 'js-validate-phone-error'.
