ok-regex

A pure JavaScript (no dependencies) solution to make HTML regex inputs self-validating

Usage

  1. Include ok-regex.js anywhere in your page, it runs on load.
  2. Add CSS class 'js-ok-regex' to all regex inputs you want validated.
  3. Optionally include ok-regex.css (or its contents) in your styles.

Limitations

This validator uses a small collection of simple regular expressions to ensure valid delimiters and modifiers, but does not deeply inspect the subject. False readings are possible. I abstained from using Javascript's own RegExp object to validate the pattern as the pattern may be intended for use in another language.

Styling Errors

Inputs with invalid regexs can be styled by applying styles to the error class. By default this class is 'js-ok-regex-error'.
