ok-creditcard

A pure JavaScript (no dependencies) solution to make HTML creditcard inputs self-validating

Usage

  1. Include ok-creditcard.js anywhere in your page, it runs on load.
  2. Add CSS class 'js-ok-creditcard' to all creditcard inputs you want validated.
  3. Optionally include ok-creditcard.css (or its contents) in your styles.

Support

  • Tested successfully in Chrome Win/Mac.
  • Tested successfully in Safari Win/Mac.

Basic Usage

The basic credit card validation functionality can be added simply by including ok-creditcard.js anywhere in your document and adding CSS class 'js-ok-creditcard' or type 'creditcard' to all inputs you want validating. Advanced Usage

By default, credit cards are checked against all known (to me) types. You can customize which cards are supported by manipulating okCreditCard.Types at any time.

Styling Errors

Inputs with invalid values can be styled by applying styles to the error class. By default this class is 'js-ok-creditcard-error'.
