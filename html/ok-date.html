Demo for ok-date.js

ok-date.js is a pure JavaScript (no dependencies) solution to make HTML date inputs self-validating.

Basic Usage

The basic date validation functionality can be added simply by including ok-date.js anywhere in your document and adding CSS class 'js-ok-date' or type 'date' to all inputs you want validating. It works like this:


Alternate Date Formats

Three parsing and output date formats are supported: Month-first (American), Year-first (ISO), and Date-first (Most of Earth). By default, dates will be output in ISO format, but if another three-part format is input, it will attempt Month-first parsing.




Additionally, a variety of (English) spoken formats are supported. Try these:

  • Now
  • Today
  • tomorrow
  • yesterday
  • 6th
  • 6th October
  • 10th Feb 2004
  • Feb 10th 2004
  • Feb 10th
  • next Friday
  • last Friday
  • 04/08/2006
  • 2004-04-04 (ISO: yyyy-mm-dd)
  • 04-04-04 (mm-dd-yy | dd-mm-yy)
  • 04-04 (mm-dd | dd-mm)
  • Friday week

Try these special keys:

  • Right Arrow - sets blank inputs to today
  • Up Arrow - increase day by one
  • Down Arrow - decrease day by one

Styling Errors

Inputs with invalid dates can be styled by applying styles to the error class. By default this class is 'js-ok-date-error'.
