ok-time

A pure JavaScript (no dependencies) solution to make HTML time inputs self-validating

Usage

  1. Include ok-time.js anywhere in your page, it runs on load.
  2. Add CSS class 'js-ok-time' to all time inputs you want validated.
  3. Optionally include ok-time.css (or its contents) in your styles.

Support

  • Tested successfully in FireFox Win/Mac.
  • Tested successfully in Chrome Win/Mac.

Basic Usage

The basic time validation functionality can be added simply by including ok-time.js anywhere in your document and adding CSS class 'js-ok-time' or type 'time' to all inputs you want validating. Some browsers do not support time inputs and will treat them as type 'text'. It works like this:


24 Hour Usage

By default, times are reformatted in the 12 hour format. CSS class 'js-ok-time-12' is an explicit alias for this default behavior. To render 24 hour times, apply css class 'js-ok-time-24' to the input. Below one of each input with a 24 hour time entered.


Try these:

  • Now
  • Noon
  • Midday
  • Midnight
  • 0530
  • 1730
  • 6am
  • 4pm
  • 04:04 (hh:mm)
  • 04:04:04 (hh:mm:ss)

Try these special keys:

  • Up Arrow - Increase one minute
  • Shift + Up Arrow - Increase one hour
  • Down Arrow - Decrease one minute hour
  • Shift + Down Arrow - Decrease one hour
  • Right Arrow - Set blank input to current time

Styling Errors

Inputs with invalid values can be styled by applying styles to the error class. By default this class is 'js-ok-time-error'.
