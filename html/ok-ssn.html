ok-ssn

A pure JavaScript (no dependencies) solution to make HTML SSN inputs self-validating

Usage

  1. Include ok-ssn.js anywhere in your page, it runs on load.
  2. Add CSS class 'js-ok-ssn' to all SSN inputs you want validated.
  3. Optionally include ok-ssn.css (or its contents) in your styles.

Support

  • Tested successfully in Chrome Win/Mac.

Basic Usage

The basic SSN validation functionality can be added simply by including ok-ssn.js anywhere in your document and adding CSS class 'js-ok-ssn' or type 'ssn' to all inputs you want validating. Stricter / Looser Usage

By default, SSNs are validated without checking High Groups limits. CSS class 'js-ok-ssn-loose' is an explicit alias for this default behavior. To require High Group checking, apply css class 'js-ok-ssn-strict' to the input. Below is such an input with a private address entered.

The implementation of SSN Randomization on June 25, 2011 eliminates the significance of the High Group List, making it ineffective for validating SSNs issued through the new random assignment method. Therefore, Strict checking is only recommended for SSNs issued after June 25, 2011. Checking adults, for example, would be ok.
