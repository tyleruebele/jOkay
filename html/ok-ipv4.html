ok-ipv4

A pure JavaScript (no dependencies) solution to make HTML ipv4 inputs self-validating

Usage

  1. Include ok-ipv4.js anywhere in your page, it runs on load.
  2. Add CSS class 'js-ok-ipv4' to all ipv4 inputs you want validated.
  3. Optionally include ok-ipv4.css (or its contents) in your styles.

Basic Usage

The basic ipv4 validation functionality can be added simply by including ok-ipv4.js anywhere in your document and adding CSS class 'js-ok-ipv4' or type 'ipv4' to all inputs you want validating. Stricter / Looser Usage

By default, IPv4 addresses are validated in a loose dotted-decimal format, allowing any address from '0.0.0.0' to '255.255.255.255'. CSS class 'js-ok-ipv4-loose' is an explicit alias for this default behavior. To require public addresses, apply css class 'js-ok-ipv4-strict' to the input. Styling Errors

Inputs with invalid values can be styled by applying styles to the error class. By default this class is 'js-ok-ipv4-error'.
